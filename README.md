# Quiz Application

# Project Description

This is a simple and interactive Quiz Application built to test users' knowledge through multiple-choice questions. The application allows users to select answers, view their score at the end, and improve their learning experience in a fun way.

##  Features

* Multiple-choice questions
* Instant answer selection
* Score calculation at the end
* User-friendly interface
* Responsive design (works on mobile & desktop)

##  Technologies Used

* HTML
* CSS
* JavaScript
