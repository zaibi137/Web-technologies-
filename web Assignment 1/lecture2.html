CSS Fundamentals

Introduction

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML. CSS separates content from presentation, allowing you to control layout, colors, fonts, and visual effects.

Key Concepts

  • Selectors: Patterns used to select and style HTML elements (element, class, ID, attribute, and pseudo-class selectors).
  • Box Model: Every HTML element is a box with margins, borders, padding, and content area.
  • Colors: CSS supports named colors, hexadecimal, RGB, RGBA, HSL, and HSLA color values.
  • Typography: Control font families, sizes, weights, line heights, and text alignment.
  • Layout: Use positioning (static, relative, absolute, fixed, sticky), floats, flexbox, and CSS grid for layouts.
  • Responsive Design: Media queries allow you to apply different styles for different screen sizes and devices.

Best Practices

Use external style sheets for maintainability. Keep selectors specific but not too specific. Use CSS variables for consistent theming. Organize stylesheets logically with comments. Minimize browser-specific prefixes by using established properties.
